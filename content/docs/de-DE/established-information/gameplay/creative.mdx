# Kreativmodus

Der Kreativemodus in Hytale ist ein Sandbox-Modus, der es Spielern erlaubt, die Welt frei zu bearbeiten.

## World Editing Tools

Der Kreativmodus beinhaltet eine Vielzahl von Tools, die Spielern helfen, die Welt zu bearbeiten. Dieser enthält eine Funktion mit welcher vergangene Änderungen rückgängig gemacht werden können.

Beachte: Die nachgenannten Abschnitte waren in der Video-Demo unbenannt.

---

### Vorformungs-Werkzeug

Ermöglicht dem Benutzer, das Gelände zu erhöhen und zu senken, um dieses genauer zu formen und vermischen. Unterstützt die meisten Funktionen der Pinsel Werkzeugs.

### Linienpinsel Werkzeug

### Auswahl-Werkzeug

Ermöglicht es dem Benutzer, Regionen der Welt auszuwählen und zu bearbeiten, indem die Auswahl von Blöcken oder eine vorgefertigte Struktur über Rotation und Extrusion transformiert wird.

#### Einstellungen

- Modus (Combo Box: Set, Wall, Fill, Replace): Beschreibung unbekannt stand vom Dezember 2025.
- Material (Ganzzahlen-Schieber): Beschreibung unbekannt Stand Dezember 2025.
- Gewicht (Ganzzahlen-Schieber): Beschreibung unbekannt Stand Dezember 2025.
- Rotation (Ganzzahl): Beschreibung unbekannt Stand Dezember 2025.

### Maßstab

Ermöglicht es dem Benutzer die Entfernungen zwischen mehreren Punkten im dreidimensionalen Raum zu messen, mit Unterstützung für mehrere Messlinien und Bearbeitung von Messpunkten.

### Ebenen pinsel-Werkzeug

Unbekannt Stand Dezember 2025.

### Einfügen Werkzeug

Ermöglicht dem Benutzer, eine kopierte Auswahl von Blöcken oder einer vorgefertigter Struktur an dem Punkt einzufügen, an dem der Spieler schaut. Die Einfügung erfolgt relativ zum Ankerpunkt der Auswahl. Ist die Auswahl eine Struktur, so kann der Ankerpunkt im Strukturen-Editor gesetzt werden. Es ist unbekannt, ob der Anker der Auswahl anderweitig gesetzt werden kann.

### Machinima

Machinima is a built-in sequence editor, allowing the user to create and edit sequences of actions to be played back in game. It includes a built-in interactive keyframe editor.

The speed of playback can be adjusted via commands.

The user may add actors for the sequence to control.

- Reference
- Camera
- Entity
- Item
- Player

## Commands

Commands are entered into the world's chat console and begin with a forward slash `/`.

- `set <block>`: allows the user to set the blocks in a selection to the specified block.
- `inventory clear`: allows the user to clear their inventory.
