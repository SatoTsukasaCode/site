O Hytale foi projetado para suportar uma variedade de métodos de entrada para acomodar diferentes preferências de jogadores e plataformas.

## Sistema de Interação

O Hytale conta com um sistema de interação poderoso que concede aos plugins a capacidade de personalizar a interpretação das atribuições de teclas e as ações que elas acionam. Esse sistema é orientado a dados, permitindo que os criadores de mods criem interações complexas por meio de recursos.

Os criadores de mods podem definir atribuições de teclas personalizadas para diversas ações, como usar habilidades ou interagir com objetos no mundo do jogo. O sistema de interação foi projetado para ser poderoso, embora exija dedicação para lidar com a complexidade dos recursos.

> Você pode usar as suas próprias interações para atribuir teclas para o que desejar. O sistema de interação é muito poderoso, mas ainda precisamos trabalhar em sua usabilidade, uma vez que os recursos podem se tornar bastante complexos e difíceis de gerenciar.
