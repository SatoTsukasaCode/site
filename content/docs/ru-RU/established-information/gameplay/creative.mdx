# Творческий режим

Творческий режим в Hytale — это режим «песочницы», который позволяет игрокам свободно редактировать мир.

## Инструменты для редактирования мира

Креативный режим включает в себя множество инструментов, помогающих игрокам редактировать мир. Он включает в себя функцию отмены действий для внесенных правок.

Обратите внимание, что разделы ниже не были подписаны в видео-демонстрации инструментов.

---

### Лазерная указка

Отображает луч из камеры игрока в направлении его взгляда.

#### Настройки

- Описание неизвестно по состоянию на декабрь 2025 года.
- Цвет лазера (Hex-код цвета): Описание неизвестно по состоянию на декабрь 2025 года.

### Инструмент «Скульптурная кисть»

Позволяет игроку поднимать и опускать рельеф, чтобы более точно моделировать и смешивать формы. Поддерживает большинство функций «Кисти для рисования»

### Инструмент «Линейная кисть»

Описание неизвестно по состоянию на декабрь 2025 года.

### Инструмент выделения

Позволяет пользователю выделять и редактировать регионы мира, преобразуя выделенные блоки или префаб с помощью перемещения, вращения и выдавливания.

#### Настройки

- Mode (Combo Box: Set, Wall, Fill, Replace): Description unknown as of December 2025.
- Material (Integer Slider): Description unknown as of December 2025.
- Weight (Integer Slider): Description unknown as of December 2025.
- Pitch (Integer): Description unknown as of December 2025.

### Ruler

Allows the user to measure distances between multiple points in 3D space, with support for multiple measurement lines and editing of measured points.

### Layer Brush Tool

Описание неизвестно по состоянию на декабрь 2025 года.

### Инструмент "Вставка"

Позволяет пользователю вставить скопированную выборку блоков или префаб в точку, на которую направлен взгляд игрока. Вставка выполняется относительно точки привязки выбранного объекта. Если выбранный объект является префабом, точка привязки может быть задана в Редакторе Префабов. Неизвестно, можно ли задать точку привязки для обычного выделения иным способом.

### Инструмент «Кисть для заливки»

Описание неизвестно по состоянию на декабрь 2025 года.

### Инструмент «Кисть для рисования»

Кисти используются для больших изменений в мире.

### Машинима

Машинима — это встроенный редактор последовательностей, позволяющий пользователю создавать и редактировать последовательности действий, которые будут воспроизводиться в игре. Он включает в себя встроенный интерактивный редактор ключевых кадров.

Скорость воспроизведения может быть скорректирована с помощью команд.

The user may add actors for the sequence to control.

- Reference
- Camera
- Entity
- Item
- Player

## Commands

Commands are entered into the world's chat console and begin with a forward slash `/`.

- `set <block>`: allows the user to set the blocks in a selection to the specified block.
- `inventory clear`: allows the user to clear their inventory.
