# Contribuindo para o Hytale Modding

Agradecemos pela contribuição da comunidade para ajudar a melhorar e expandir a documentação do Hytale Modding. Seja corrigindo erros de digitação, adicionando novos tutoriais ou fornecendo exemplos de código, suas contribuições são valiosas.

## Como contribuir

### Relatando problemas

Se você encontrou um problema com a documentação, reporte-o abrindo uma nova issue em nosso [repositório do GitHub](https://github.com/HytaleModding/site/issues).

Havendo disponibilidade para resolver esse problema, é possível solicitar uma atribuição a ele e realizar o envio de um pull request.

### Enviando pull request

1. Faça um fork no repositório do GitHub;
2. Clone seu fork para sua maquina local;
3. Crie uma branch para as suas alterações.

```bash
git checkout -b my-feature-branch
```

4. Faça as suas alterações e monte um commit com mensagens descritivas sobre elas. Seguimos os [commits convencionais](https://www.conventionalcommits.org/en/v1.0.0/), então seu pull request não será aceito se as suas mensagens nos commit não seguirem a formatação correta.

```bash
git commit -m "Adicione um recurso à documentação"
```

5. Envie suas alterações para o seu fork.

```bash
git push origin my-feature-branch
```

6. Abra um pull request no repositório original e descreva suas alterações com o motivo delas poderem ser mescladas ao código.
