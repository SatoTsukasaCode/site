# Modo Creativo

El modo creativo en Hytale es un modo sandbox que permite a los jugadores editar el mundo libremente.

## Herramientas de Edición de Mundo

El modo creativo incluye una variedad de herramientas para ayudar a los jugadores a editar el mundo. Incluye la funcionalidad de deshacer para ediciones.

Tenga en cuenta que las secciones de abajo no fueron nombradas en la demostración de vídeo de las herramientas.

---

### Puntero láser

Muestra un rayo de la cámara del jugador en la dirección que el jugador está mirando.

#### Configuración

- Duration (Integer Slider): Description unknown as of December 2025.
- Laser Color (Hex Color Code): Description unknown as of December 2025.

### Herramienta de Pincel de Esculpir

Permite al usuario subir y bajar el terreno a formas más precisas de esculpido y mezclado. Soporta la mayoría de las características del Pincel

Desconocido en diciembre de 2025.

### Herramienta de selección

Permite al usuario seleccionar y editar regiones del mundo transformando selecciones de bloques o prefab mediante posición, rotación y extrusión.

#### Configuración

- Mode (Combo Box: Set, Wall, Fill, Replace): Description unknown as of December 2025.
- Material (Integer Slider): Description unknown as of December 2025.
- Weight (Integer Slider): Description unknown as of December 2025.
- Pitch (Integer): Description unknown as of December 2025.

### Regla

Permite medir distancias entre múltiples puntos en espacio 3D, con soporte para múltiples líneas de medición y edición de puntos medidos.

### Layer Brush Tool

Desconocido en diciembre de 2025.

### Herramienta de Pegar

Permite al usuario pegar una selección copiada de bloques o un prefab en el punto en el que el jugador está mirando. La pasta es relativa al punto de anclaje de la selección. Si la selección es un prefab, el punto de anclaje puede establecerse en el Editor de Prefab. Se desconoce si el ancla de la selección se puede establecer de otra manera.

### Flood Brush Tool

Unknown as of December 2025.

### Machinima

Machinima is a built-in sequence editor, allowing the user to create and edit sequences of actions to be played back in game. It includes a built-in interactive keyframe editor.

The speed of playback can be adjusted via commands.

The user may add actors for the sequence to control.

- Reference
- Camera
- Entity
- Item
- Player

## Commands

Commands are entered into the world's chat console and begin with a forward slash `/`.

- `set <block>`: allows the user to set the blocks in a selection to the specified block.
- `inventory clear`: allows the user to clear their inventory.
