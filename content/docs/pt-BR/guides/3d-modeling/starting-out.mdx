Por favor, complete o guia [Getting Started](./getting-started.mdx) antes disso.

Uma vez que você tenha completado o guia Getting Started, você deve ter conseguido um Hytale Prop model pronto para uso. Nesse guia, nós vamos cobrir como criar um modelo usando Blockbench.

You probably should change your view based on how you prefer by using the joystick at the bottom right corner of the viewport. It'll look like a joystick. Left click and rotate.

![Blockbench — Eixos](https://cdn.internal.hytalemodding.dev/assets/guides/blockbench_joystick.png)

Vamos pôr um cubo no viewport agora!

Clique no botão "**+**" que fica na seção "OUTLINER" e clique em "**Add Cube**" no menu suspenso.

Agora você deve estar vendo algo assim:

![Blockbench — Adição de cubos](https://cdn.internal.hytalemodding.dev/assets/guides/blockbench_cube.png)

You will see some values populate the "Transform" field. This is how you can control how big the cube is in all axis's

É assim que você consegue adicionar mais cubos e outras formas ao seu modelo.
