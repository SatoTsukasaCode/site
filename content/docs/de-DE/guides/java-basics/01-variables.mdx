Variablen sind Container, die Daten in Ihrem Programm speichern. Stellen Sie sich diese als beschriftete Boxen vor, in denen Sie verschiedene Arten von Informationen legen können.

## Was ist eine Variable?

Eine Variable hat drei Hauptkomponenten:

1. **Typ**: Welche Art von Daten sie enthält (z. B. Integer, String, Boolean)
2. **Name**: Die Bezeichnung, die Sie verwenden, um auf die Variable zu verweisen
3. **Wert**: Die darin enthaltenen Daten

## Deklarieren/Erstellen von Variablen

In Java müssen Sie eine Variable deklarieren, bevor Sie sie verwenden, so wie Sie eine Box benötigen, bevor Sie etwas in sie ablegen können. Die Syntax lautet:

```java
int alter;
alter = 18;

int alter = 18; // Deklaration und Zuweisung in einer Zeile
```

## Primitive Datentypen

Sie fragen sich vielleicht, was "int" bedeutet. Es ist ein primitiver Datentyp in Java. Java hat 8 primitiven Datentypen. Hier sind die wichtigsten für Modding:

| Datentyp | Beschreibung                       | Beispiel                   |
| -------- | ---------------------------------- | -------------------------- |
| int      | Ganze Zahlen                       | `int punkte = 100;`        |
| double   | Dezimalzahlen                      | `double pi = 3.14;`        |
| boolean  | Wahr oder falsch (true oder false) | `boolean istAktiv = true;` |
| char     | Einzelnes Zeichen                  | `char note = 'A';`         |
| String   | Zeichenfolge (Text)                | `String name = "Hytale";`  |

## Typumwandlung

Manchmal möchten Sie vielleicht nicht, dass eine Variable den gleichen Typ beibehält. Sie können zwischen den Typen umwandeln:

### Automatisch

```java
int num = 10;
double decimalNum = (double) num; // 10.0 - funktioniert automatisch
```

### Manuell (Casting)

```java
double decimalNum = 9.78;
int num = (int) decimalNum; // 9 - Nachkommastellen fallen weg
```

## Konstanten

Wenn Sie eine Variable wollen, die sich nicht ändern sollte, verwenden Sie das `final` Schlüsselwort:

```java
final double PI = 3.14159;
```

**Konvention**: Konstanten verwenden NUR_GROßBUCHSTABEN_MIT_UNTERSTRICHEN.
