Hytale está diseñado para soportar una variedad de métodos de entrada para acomodar diferentes plataformas y estilos de juego.

## Sistema de Interacción

Hytale cuenta con un robusto sistema de interacciones que permite a los plugins personalizar cómo se interpretan los atajos de teclado y las acciones que activan. El sistema está basado en datos, habilitando a los modders el crear interacciones complejas a través de recursos.

Modders pueden definir atajos de teclado personalizados para varias acciones, como el usar habilidades o interactuar con objetos en el mundo. El sistema de interacción está diseñado para ser poderoso, aunque puede requerir un esfuerzo para mantener la complejidad entre los recursos involucrados

> Puedes usar tus propias interacciones para usar tus atajos de teclado de la forma que quieras. El sistema de interacciones es muy poderoso pero aún necesita trabajo en su usabilidad, ya que los recursos pueden volverse complejos y difíciles de mantener
