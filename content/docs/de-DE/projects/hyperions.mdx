# Hyperions

<Callout>
  Dies ist ein Community-geschaffenes Projekt von
  [MaeMachineBroke](https://github.com/maemachinebroke).
</Callout>

**Hyperions** ist eine API-Bibliothek für Hytale, die darauf ausgelegt ist, anderen Mods eine API zur Speicherung und Übertragung von Energie zur Verfügung zu stellen. Die Bibliothek verwendet die Transfer-API von Community Mitglied Shailist {/* TODO Link */}, und basiert auf der [Energy](https://github.com/TechReborn/Energy) API vom TechReborn Team.

Die Standardeinheit der Energie, die Hyperions verwendet, wird Hyperion (kurz `Hy` genannt), und ein Hyperion entspricht in etwa {/* TODO */}. Die Standardeinheit der Transferrate ist der Hyperion pro Tick (`Hy/t`) oder alternativ durch 30 Teilen, um Hyperions pro Sekunde zu bekommen (`Hy/s`).
