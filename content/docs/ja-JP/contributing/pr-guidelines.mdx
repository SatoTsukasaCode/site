# プルリクエストのガイドライン

スムーズで効率的なレビューを行うために、コントリビューターとレビュアーは、プルリクエスト（以下、PRという）を提出・レビューする際に次のガイドラインに従う必要があります。

## コントリビューター

1. **明瞭なタイトルと説明**
   -
   - PRで行うこと、変更が必要な理由、および関連する情報を詳しく説明してください。
2. **Issueへのリンク**
   - 関連するIssueとPRをリンクさせるために、PRのタイトルに GH-ISSUE-NUMBER を追加してください。 これはすべてのPRに適用されます。
