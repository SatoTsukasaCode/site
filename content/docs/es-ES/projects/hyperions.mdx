# Hyperions

<Callout>
  Este es un proyecto comunitario hecho por el usuario [MaeMachineBroke](https://github.com/maemachinebroke).
</Callout>

**Hyperions** es una libreria API para Hytale, la cual está diseñada para proveer otros mods con una API para el almacenaje, transmisión y recepción de energía. La librería usa la Transfer API gracias a miembros de la comunidad, como shailist {/* TODO link */}, quien se basó en la API de [Energy](https://github.com/TechReborn/Energy) por el grupo de TechReborn.

La unidad fundamental de la energía usada por Hyperions es llamada Hyperion (abreviado como Hy), y uno de estos equivale aproximadamente a {/* TODO */}. La unidad fundamental de la velocidad de transferencia es el Hyperion por cierto tiempo (Hy/t), o de forma alternativa, dividir por 30 para obtener Hyperion por segundo (Hy/s)
