Willkommen zu deiner Reise in die Java-Programmierung! Dieser Leitfaden soll dir die Grundlagen von Java näher bringen.

## Was ist Java?

Java ist eine leistungsstarke, objektorientierte Programmiersprache, die 1995 geschaffen wurde. Sie ist eine der beliebtesten Programmiersprachen der Welt und wird für alles von Handy-Apps bis hin zu Unternehmenssoftware verwendet. Am wichtigsten für uns ist aber, dass es die Sprache ist die Hytale für ihre Server & Modding API verwendet.

## Was du lernen wirst

Diese Anleitung ist so strukturiert, dass du Schritt für Schritt durchgeführt wirst:

1. Java Syntax und Grundlagen
2. Ablaufsteuerung
3. Objektorientierte Programmierung

Wir werden uns dann nur auf Hytal-spezifische Konzepte konzentrieren. (tbd)

## Umgebung einrichten

Bevor wir in die Programmierung eintauchen können, musst du ein paar notwendige Dinge installieren:

### Java Development Kit (JDK)

Lade dir JDK 25 von der [offiziellen Oracle Website](https://www.oracle.com/java/technologies/javase-jdk25-downloads.html) herunter oder verwenden eine Open-Source Alternative wie [AdoptOpenJDK](https://adoptopenjdk.net/) und installiere diese.

### Integrierte Entwicklungsumgebung (IDE)

Eine IDE ist eine Software-Anwendung, die Programmierern umfassende Möglichkeiten für die Softwareentwicklung bietet. Beliebte Optionen für Java beinhalten:

* [IntelliJ IDEA](https://www.jetbrains.com/idea/)
* [Eclipse](https://www.eclipse.org/)

1. Lese jeden Artikel nach der Reihenfolge - Konzepte bauen aufeinander auf

## Die Modding-Mentalität

Modding ist kreatives Problemlösen. Du wirst:

* Neue Blöcke, Items und Kreaturen zu Hytale hinzufügen
* Die Spielmechaniken ändern, um einzigartige Erlebnisse zu schaffen
* Systeme erstellen, die mit der Spielwelt interagieren
* Deine Kreationen mit der Community teilen
