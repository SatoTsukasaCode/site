# HytaleModdingに貢献する

HytaleModding のドキュメントを改善・拡張するため、コミュニティからの貢献を歓迎しています。 誤字の修正や新しいチュートリアルの追加、コード例の提供など、どのようなことでも感謝いたします。

## どのようにして貢献する？

### 問題を報告する

ドキュメントに問題を見つけた場合は、 [GitHub](https://github.com/HytaleModding/site/issues)で新しいissueを作り報告してください。

issueを自分で解決できる場合は、プルリクエストを提出することもできます。

### プルリクエストの提出

1. GitHubでリポジトリをフォークします。
2. フォークしたリポジトリをローカル環境に複製します。
3. 変更用に新しいブランチを作成します。

```bash
git checkout -b my-feature-branch
```

4. 変更した内容をわかりやすくメッセージにまとめてコミットする。 私たちは[conventional commits](https://www.conventionalcommits.org/en/v1.0.0/)の仕様に従っているため、プルリクエストがこの仕様に則っていない場合、受け付けることはできません。

```bash
git commit -m "Add feature X to documentation"
```

5. 自分のフォークにプッシュします。

```bash
git push origin my-feature-branch
```

6. 元のリポジトリにプルリクエストを作成し、加えた変更内容と、その変更をマージするべき理由を説明します。
